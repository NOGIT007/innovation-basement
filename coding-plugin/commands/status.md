Display current workflow status.

## Status Report

1. **Current Phase**
   - Phase name and description
   - Associated GitHub issue
   - Completion percentage

2. **Context Health**
   - Estimated context usage
   - Warning if >60%
   - Recommendation if >80%

3. **Bug Fix Tracking**
   - Number of consecutive failed attempts
   - Warning if >3 attempts

4. **Git Status**
   - Current branch
   - Uncommitted changes
   - Recent commits

5. **Recommendations**
   - Next actions
   - Warnings or concerns
